#Conclusiones

Esta tarea me ha permitido aplicar de forma práctica los conceptos básicos de la unidad: uso de Markdown, control de versiones con Git, automatización con GitHub Actions y despliegue de servicios mediante Docker. Gracias a ello, he comprendido mejor cómo se integran estas herramientas dentro de un flujo de trabajo tipo SecDevOps, desde la creación del repositorio hasta la publicación de la documentación con GitHub Pages.

Además, la creación del contenedor NGINX me ha ayudado a reforzar el uso de volúmenes y la ejecución de servicios en entornos aislados. En conjunto, considero que esta actividad ha sido útil para afianzar conocimientos fundamentales que servirán en futuras prácticas y proyectos del módulo.
